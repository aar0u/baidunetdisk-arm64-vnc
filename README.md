# baidunetdisk-arm64-vnc

基于百度网盘官方Linux版本，安装在docker容器内，支持通过noVNC或者VNC访问。
支持 树莓派4B arm64版本系统，理论上也支持其他arm64的设备。

## 版本

|:-|:-|:-|
